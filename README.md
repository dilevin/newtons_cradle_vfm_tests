# newtons_cradle_vfm_tests
A bunch of generative videos of Newton's Cradle

The videos are stored using [GIT LFS](https://git-lfs.com). Please make sure this is installed before cloning the repo.

Occasionally the full files will fail to download. If this happens, try the following in the repo directory to clear the lfs cache:
```
rm -rf .git/lfs/objects
git lfs fetch
```
You don't need to checkout the repo to view the files, just go to the **veo** directory, click on each file and then click **view raw**
